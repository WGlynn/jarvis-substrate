**[F·mechanism-deployment-as-comeback]**

## ⚙ Rule

∀ hostile / unproductive / drive-by actor ⇒
- ✗ emotional engagement
- ✗ pile-on response
- ✗ ban-debate ∨ moderation-fight
- ✓ run classifier (axis-1 substance-floor)
- ✓ log to public contribution-graph w/ commit hash
- ✓ optionally send actor the commit URL ≡ "receipt"
- ✓ null-player axiom auto-zeros attribution mass

⇒ structure-does-the-work @ comeback-layer
⇒ posture ≡ "I'm not mad, here's your honest receipt"
