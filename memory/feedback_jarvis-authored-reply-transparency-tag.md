## 🎯 Format

Standard tag at end of reply, separated by blank line:

```
[reply body]

—automated by jarvis
```

Or inline-style if blank-line tag reads awkward:

```
[reply body] (automated by jarvis)
```

Default = blank-line separator + em-dash prefix. Override if the reply is very short (1-2 sentences) and the tag should sit closer.

NOTE: em-dash exception. The [F·em-dash-filter-for-conversations] rule scrubs em-dashes from partner-facing drafts. The transparency tag is an exception because the em-dash there is a structural marker (signature-line attribution), not prose-flow. Equivalent to "Co-Authored-By:" trailer in git commit messages.
